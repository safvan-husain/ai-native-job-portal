# Data Directory

Place your sample documents here to build your knowledge base.

## Supported Formats

- **PDF**: `.pdf` files
- **Text**: `.txt` files
- **Word**: `.docx` files
- **Markdown**: `.md` files
- **HTML**: `.html` files

## Sample Data Ideas

1. **Personal documents**: Research papers, articles, notes
2. **Public datasets**: Wikipedia dumps, arXiv papers
3. **Documentation**: Python docs, framework guides
4. **Books**: Project Gutenberg texts
5. **Your own content**: Blog posts, essays, reports

## Structure

```
data/
├── sample.txt          # Example text file
├── articles/           # Collection of articles
├── documentation/      # Technical docs
└── books/             # E-books or book chapters
```

## Tips

- Start with 5-10 small documents to test your RAG system
- Ensure you have rights to use the documents
- Organize by topic or source for easier management
- Keep raw documents separate from processed data
