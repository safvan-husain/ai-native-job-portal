# Jupyter Notebooks

Interactive notebooks to learn RAG step-by-step.

## Recommended Order

1. **01_embeddings_intro.ipynb**: Learn about text embeddings
2. **02_vector_db_basics.ipynb**: Store and search vectors
3. **03_document_processing.ipynb**: Load and chunk documents
4. **04_basic_rag.ipynb**: Build your first RAG system
5. **05_advanced_rag.ipynb**: Optimization and advanced techniques
6. **06_evaluation.ipynb**: Measure RAG performance

## Getting Started

```bash
# Install Jupyter
pip install jupyter

# Start Jupyter
jupyter notebook

# Or use JupyterLab
jupyter lab
```

## Tips

- Run cells sequentially
- Experiment with different parameters
- Save your findings in markdown cells
- Create your own notebooks for specific experiments

## Alternative: VS Code

You can also run these notebooks in VS Code with the Jupyter extension.
